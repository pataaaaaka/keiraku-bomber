# 経絡ボンバー - シンプル版

**確実に動作する最小構成版**

レスポンシブ機能とPWA機能を削除し、ゲーム本体のみに特化したバージョンです。

## 特徴

- ✅ 確実に動作（余計な機能なし）
- 🎮 完全なゲーム機能
- 💻 PC向けキーボード操作
- 🔥 5つの臓器ステージ
- 💊 生薬システム

## セットアップ

```bash
npm install
npm run dev
```

## デプロイ

```bash
# GitHubにプッシュ
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/keiraku-bomber.git
git push -u origin main

# Vercelで接続
```

## 操作方法

- 移動: ↑↓←→
- お灸: SPACE
- 鍼: Z(↑) X(↓) C(←) V(→)

## トラブルシューティング

問題が発生した場合：

1. `npm install` を再実行
2. ブラウザのキャッシュをクリア
3. Vercelでキャッシュをクリアして再デプロイ
